Overview

Posl provides a framework to implement a DSL with the following properties.
  • single pass
  • procedural
  • interpreted
  • prefix based
Within that scope, you may add, remove, and define capabilities to provide a DSL that does exactly what you need it to do.

Available libraries

Posls core library is posl-engine.jar, this library provides all the information necessary for you to implement a DSL.

A sample language is also provided in the po-lang.jar
